
Before starting to work on the project, make sure that you have Ionic CLI installed. To do this, run the installation script below:

For Linux/Mac systems:

./install.sh

For Windows systems:

install.bat

Dla systemów Linux/Mac:

./install.sh

Dla systemów Windows:

install.bat
